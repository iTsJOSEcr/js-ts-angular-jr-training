# JavaScript / TypeScript / Angular Jr Training

Repositorio de práctica para reforzar JavaScript, TypeScript y Angular siguiendo un plan de entrenamiento de 6 semanas.

## Objetivo

Desarrollar habilidades prácticas de programación, buenas prácticas, uso de Git y preparación para trabajar en proyectos Angular.

---

# Semana 1 - Fundamentos de JavaScript

## Día 1

### Temas vistos

- Variables (`let` y `const`)
- Funciones
- Operaciones matemáticas
- Condicionales
- Lógica básica

### Archivos

- `01-operaciones-basicas.js`
- `02-calculos-tienda.js`
- `03-condicionales.js`
- `04-logica-basica.js`

### Aprendizajes

- Creación de funciones reutilizables.
- Uso de operadores matemáticos y lógicos.
- Uso de condicionales.
- Organización del código en archivos.
- Flujo básico de Git:
  - Branches
  - Commits
  - Push
  - Pull Request
